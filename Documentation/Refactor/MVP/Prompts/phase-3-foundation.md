# Phase 3: Foundation & Reset

> Build the structural CSS that everything depends on: reset, foundation tokens (Tier 3), and utility classes.

---

## Context

Read these specs before starting:

## Deliverables

### 3.1 -- Reset (`src/reset.css`)

This file MUST contain the layer order declaration as its first meaningful line:

```css
@layer reset, primitives, semantic, component, utilities;
```

Then wrap all reset rules in `@layer reset { }`:

**Reset rules**:
- `*, *::before, *::after { box-sizing: border-box; }`
- `* { margin: 0; padding: 0; }`
- `html { scroll-behavior: smooth; scroll-padding-top: 80px; }` (80px for sticky nav)
- `body` defaults: font-family, line-height, color, background from tokens
- `img, picture, video, canvas, svg { display: block; max-width: 100%; }`
- `:focus-visible { outline: 2px solid var(--focus-ring); outline-offset: 2px; }`
- `prefers-reduced-motion` global gate:
  ```css
  @media (prefers-reduced-motion: reduce) {
    *, *::before, *::after {
      animation-duration: 0.01ms !important;
      animation-iteration-count: 1 !important;
      transition-duration: 0.01ms !important;
      scroll-behavior: auto !important;
    }
  }
  ```

### 3.2 -- Foundation (`src/foundation.css`)

All rules wrapped in `@layer component { }` (Tier 3 tokens live in the component layer):

**Typography**:
- Font families: `--font-sans: 'Inter', system-ui, sans-serif;` and `--font-mono: 'JetBrains Mono', monospace;`
- Fluid type scale: `--step--2` through `--step-5` using `clamp()` formulas
- Fixed UI text scale: `--ui-text-2xs` (10px) through `--ui-text-xl` (20px)

**Spacing**:
- `--space-1` (4px) through `--space-20` (80px)
- Includes `--space-1-5` (6px) for half-step

**Radius**:
- `--radius-sm` (10px), `--radius-md` (16px), `--radius-lg` (24px), `--radius-xl` (32px), `--radius-full` (9999px)

**Shadows**:
Shadow token values are defined in `tokens.css` (generated by CSS emitter). Foundation references them via var().

**Control Heights**:
- `--control-sm` (32px), `--control-md` (36px), `--control-lg` (44px), `--control-xl` (56px)

**Motion Tokens**:
- Timing: `--motion-instant` (100ms), `--motion-fast` (160ms), `--motion-base` (240ms), `--motion-slow` (360ms), `--motion-deliberate` (500ms)
- Easing: 5 cubic-bezier curves (`--ease-out`, `--ease-bounce`, `--ease-smooth`, `--ease-slam`, `--ease-elastic`) + 2 `linear()` spring curves (`--spring-snappy`, `--spring-gentle`)

**Z-Index**:
- `--z-base` (1), `--z-nav` (100), `--z-modal` (200), `--z-toast` (300), `--z-tooltip` (400)

**Container Widths**:
- `--container-sm` (640px), `--container-md` (768px), `--container-lg` (1024px), `--container-xl` (1280px)

**Component Token Aliases**:
- Button, badge, toggle, chart token mappings that reference semantic tokens

### 3.3 -- Utilities (`src/utilities.css`)

All rules wrapped in `@layer utilities { }`:

Utility classes (~28 total):
- Flex: `.flex`, `.flex-col`, `.flex-wrap`, `.items-center`, `.justify-center`, `.justify-between`
- Gap: `.gap-1` through `.gap-6`
- Spacing: `.mt-*`, `.mb-*`, `.mx-auto`, `.p-*`
- Text: `.text-center`, `.text-left`, `.text-right`, `.text-sm`, `.text-xs`, `.font-bold`, `.font-mono`
- Width: `.w-full`, `.max-w-*`
- Display: `.hidden`, `.block`, `.inline-flex`

## Acceptance Criteria

- [ ] `@layer reset, primitives, semantic, component, utilities;` is the first rule in reset.css
- [ ] `prefers-reduced-motion` gate exists in reset.css
- [ ] Foundation tokens are all in `@layer component`
- [ ] Utility classes are all in `@layer utilities`
- [ ] A blank HTML page importing tokens + reset + foundation + utilities renders correctly
- [ ] All custom properties resolve (no undefined var() references)
- [ ] Font families, spacing, radius, and z-index values match specs
