Toast Notifications

Spring-powered entry, stack compression, swipe-to-dismiss, and auto-dismiss with progress.

Animation Reference

Entry: y: 80→0, opacity 0→1, spring (500/30)
Stack: existing toasts shift up, spring (400/35)
Dismiss: swipe x>100px exits, else spring back (500/30)
Auto-dismiss: 5s progress bar, then slide-right exit
