Delightful

A warm design system for editors, terminals, notes, agents, and the messy human work between them. Built from a three-tier token architecture and shipped across the tools I actually live in.

Design System Components, tokens, interactions, composition, and implementation notes. Color OKLCH primitive families, semantic roles, and component bindings. Motion CSS animation catalog with reduced-motion behavior. Animation Spring physics, FLIP, particles, gestures, and motion experiments.

A system with fingerprints

Delightful uses warm neutrals, unapologetic borders, solid offset shadows, and bright-but-controlled accents. The goal is not generic polish. It is software that feels sturdy, expressive, and a little more alive.
