# Delightful Starship

Starship prompt theme using the Delightful color system. Based on the [Gruvbox Rainbow](https://starship.rs/presets/gruvbox-rainbow) Starship preset, re-themed with the Delightful color system.

## Install

1. **Install Starship** — [starship.rs](https://starship.rs)
2. **Install Nerd Font**
   ```sh
   brew install --cask font-jetbrains-mono-nerd-font
   ```
3. **Copy config**
   ```sh
   cp starship.toml ~/.config/starship.toml
   ```

## Light Mode

A light palette variant is included as a commented block at the bottom of `starship.toml`. To use it, swap `palette = 'delightful_dark'` to `palette = 'delightful_light'` and uncomment the light palette block.
